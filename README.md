# Orchardia Online V1.4 — Inventory & Shop

ต่อยอดจาก V1.3 และคงระบบมือถือทั้งหมด

## เพิ่มใหม่
- กระเป๋า Inventory แบบเปิด/ปิดได้
- แสดงพลังโจมตี HP สูงสุด ยา HP และเมล็ดหายาก
- ใช้ยา HP เพื่อฟื้นพลังชีวิต
- อัปเกรดธนูได้ถึงระดับ 5
- พลังโจมตีเพิ่มตามระดับอาวุธ
- เพิ่มพ่อค้าเมล็ดพันธุ์ในหมู่บ้าน
- ร้านค้าขายยา HP และเมล็ดหายาก
- เพิ่มปุ่มโต้ตอบ คุย / ร้านค้า / เปิดหีบ
- NPC ไม่ส่งเควสต์อัตโนมัติ ต้องกดคุย
- หีบสมบัติต้องกดเปิด
- ไอเทมดรอปแยกเป็นเหรียญ ยา HP และเมล็ดหายาก
- บันทึก Inventory ระดับอาวุธ และสถานะเควสต์
- หยุดตัวละครอัตโนมัติเมื่อเปิดหน้าต่าง UI
- ยังเป็น Flat Project 4 ไฟล์สำหรับ GitHub Mobile
