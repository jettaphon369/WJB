# Orchardia Online — Flat GitHub Version

ไฟล์ทั้งหมดอยู่ระดับเดียวกัน ไม่มีโฟลเดอร์ย่อย

## ไฟล์
- index.html
- style.css
- game.js
- README.md

## วิธีอัป GitHub จากมือถือ
1. เข้า Repository
2. Add file → Upload files
3. เลือกทั้ง 4 ไฟล์พร้อมกัน
4. Commit changes
5. เปิด Settings → Pages
6. Deploy from branch → main → /root

## ระบบที่มี
- WebGL 3D แบบไม่ใช้ CDN
- กล้องบุคคลที่สาม
- จอยมือถือ
- โจมตี / สกิล / กลิ้ง
- ตั๊กแตนและด้วง
- เควสต์ 0/5
- บอสหลังจบเควสต์
- HP / MP / XP / Level
- เงิน ไอเทม และ localStorage

สำคัญ: ต้องตั้งชื่อไฟล์หลักว่า `index.html`
