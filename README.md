# Orchardia Online V1.5 — Visual & Combat Polish

ต่อยอดจาก V1.4 โดยไม่รื้อระบบเดิม

## เพิ่ม/แก้
- Canvas รองรับพื้นที่ Browser มือถือด้วย visualViewport
- ลดปัญหาพื้นที่สีฟ้าด้านล่าง
- กล้องลื่นขึ้นด้วย Camera Smoothing
- เพิ่มระบบ Lock Target
- เพิ่มวงแหวนใต้เป้าหมาย
- เพิ่ม Skill 2 แบบ AOE รอบตัว
- แสดง Cooldown บนปุ่มสกิล
- ปุ่มคุย/ร้านค้า/เปิด แสดงเฉพาะเมื่ออยู่ใกล้สิ่งที่โต้ตอบได้
- ปรับ HUD และปุ่มมือถือให้กะทัดรัดขึ้น
- คง Inventory / Shop / Quest / NPC / Save จาก V1.4
- คงระบบป้องกัน Double Tap Zoom / Pinch Zoom / Safe Area
