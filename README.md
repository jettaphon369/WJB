# Orchardia Online V1.2 — Visual World

ต่อยอดจาก V1.1 โดยคงการรองรับมือถือทั้งหมด

## เพิ่มแล้ว
- ปรับมุมกล้องให้เห็นโลกชัดขึ้น
- เกาะลอยฟ้ามีชั้นดินและโครงสร้างมากขึ้น
- เพิ่มถนนดินกลางแผนที่
- เพิ่มหมู่บ้าน บ้าน และหลังคา
- เพิ่มบ่อน้ำและสะพานไม้
- เพิ่มต้นไม้ หิน ดอกไม้ และของตกแต่ง
- ปรับ Seed Ranger ให้มีรูปร่างอ่านง่ายขึ้น
- ปรับตั๊กแตนและด้วงให้มีรูปร่างเฉพาะ
- เพิ่มแถบ HP เหนือศัตรู
- เพิ่มหางแสงของกระสุน
- คงระบบป้องกันซูม Double Tap และ Pinch Zoom บนมือถือ
